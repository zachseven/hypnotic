# Changelog

All notable changes to hypnotIC will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025

### Added
- Initial release
- Complete interpreter with lexer, parser, and interpreter
- REPL mode for interactive coding
- File execution mode
- Debug mode (`-d` flag)
- One-liner execution (`-c` flag)

### Language Features
- Variables and assignment
- Print statement (`*`)
- Input (`PV`)
- Conditionals (`?`, `file`, `Ls`, `IC`)
- For loops (`4 i n 10`)
- While loops (`wyl`)
- Functions (`fun`, `re:`)
- Lambda expressions (`\`)
- Arrays (`rayz`, `up`, `!!!`, `"`)
- Dictionaries
- Try/catch (`hard`, `C`)
- Imports (`->`)
- Comments (`fuck`)
- All comparison operators (`iz`, `aint`, `>`, `<`, `>=`, `<=`)
- All logical operators (`&`, `or`, `RHO`, `-`)
- All math operators (`+`, `-`, `*`, `/`, `^`, `mold`)
